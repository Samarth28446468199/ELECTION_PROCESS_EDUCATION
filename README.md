# Election Process Education Website

> **Hackathon 2026 — Civic Tech Track**  
> A fully 3D, interactive, animated website that educates citizens on how elections work.

## 🌟 Features

- **3D Animated Ballot Box** with CSS `transform-style: preserve-3d`
- **Custom Mouse Cursor** with trailing effect
- **Mouse-Tilt Cards** — every card reacts to hover in 3D
- **Interactive Quiz** — 6 civic knowledge questions with scoring
- **Full Election Process Timeline** — 6 steps with scroll animations
- **Myth vs Fact** section busting common election misconceptions
- **Voting Systems Explainer** — FPTP, Proportional, Ranked-Choice, Two-Round
- **Types of Elections** — National, State, Local, Referendum, Primary, By-election
- **Hero Parallax** effect on mouse movement
- **Fully Responsive** — Mobile friendly
- **Smooth Scroll + Scroll Animations**
- **Dark Space Theme** with glowing orbs and grid overlay

## 🛠️ Tech Stack

- Pure HTML5 + Vanilla CSS + Vanilla JavaScript
- Google Fonts (Outfit + Space Grotesk)
- Zero dependencies — no frameworks, no build tools

## 🚀 Run Locally

Just open `index.html` in any modern browser. No server needed.
